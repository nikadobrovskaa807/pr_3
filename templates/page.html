  • manage.py

    manage.py: выполняет различные команды проекта, например, создает и запускает приложение.

  • Название проекта

    Название проекта - папка проекта, которая содержит некоторые файлы, они будут представлены ниже.

  • Файлы, которые содержатся в папке "Название проекта"

    __init__.py: данный файл указывает, что папка, в которой он находится будет рассматриваться как модуль;

    setting.py: содержит настройки конфигурации пакета;

    urls.py: содержит шаблоны URL-адресов, по сути определяет маршрутизацию страницы;

    wsgi.py: содержит свойства конфигурации WSGI. Он используется при развертывании проекта;

    asgi.py: Asynchronous Server Gateway Interface расширяет возможности WSGI, добавляя поддержку для взаимодействия между асинхронными веб-сервисами и приложениями.
